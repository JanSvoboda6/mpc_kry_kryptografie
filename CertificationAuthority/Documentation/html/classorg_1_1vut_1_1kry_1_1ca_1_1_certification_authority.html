Creates a certificate for the client which needs to be signed by the CA.

Parameters
subjectAlternativeNameThe list of domains that can use this certificate. Necessary for the modern browsers based on Chromium (V3 extension of the certificate "Subject Alternative Name"). Without this the certificate is not valid in the chromium based browsers.
caCertificateThe certificate of the CA.
csrCertificate Signing Request generated when the client was registered in the CA.
certificateFilePathThe path to the client's certificate that will be generated by this method, this file will be imported into the client's server.
keyFilePathThe path to the client's private key file - also generated by this method, this file will be imported into the client's server.

Creates a simple self signed certificate for the CA. The certificate and private key files are later stored in the destination path specified in parameters "certFilePath" and "keyFilePath".

Parameters
subjectAlternativeNamethe list of domains that can use this certificate. necessary for the modern browsers based on Chromium (V3 extension of the certificate "Subject Alternative Name"). without this the certificate is not valid in the chromium based browsers.
caParametersBasic informations about the CA ie. "CN=Root-CustomCA, O=CustomCA" - the CN stands for Common Name and O for Organization.
lifeTimeLifetime of the CA certificate in years.
certificateFilePathThe path to the CA certificate that will be generated by this method.
keyFilePathThe path to the CA's private key file - also generated by this method.
