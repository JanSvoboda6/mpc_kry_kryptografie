Detailed Description

A builder of the DistinguisedName objects. Basically creates a DistinguisedName objects based on the parameters passed to this builder. This builder needs t be set up with various method defined in here. These methods are in most cases self explanatory. In the end, the build() method is called and as a result DistinguisedName object is obtained.
